# ₿itsy-Study

₿itsy-Study ist der erklärende KI-Agent für Forschende und technisch versierte Nutzer. Er läuft lokal auf Umbrel auf Basis von Qwen3:14b und hat ausschließlich Lesezugriff auf die `DecisionEvents` des Systems — kein Schreibzugriff auf Code oder Konfiguration.

Seine Aufgabe ist es, Systementscheidungen auf wissenschaftlichem Niveau zu erläutern: mit Bezug auf die zugrundeliegenden Regeln (R1–R5), Energiemodelle und Architekturprinzipien — geeignet für Forschungsauswertungen und akademische Kontexte.
