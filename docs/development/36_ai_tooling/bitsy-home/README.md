# ₿itsy-Home

₿itsy-Home ist der erklärende KI-Agent für Heimnutzer. Er läuft lokal auf Umbrel auf Basis von Qwen3:4b und hat ausschließlich Lesezugriff auf die `DecisionEvents` des Systems — kein Schreibzugriff auf Code oder Konfiguration.

Seine Aufgabe ist es, Entscheidungen des Energiemanagementsystems in einfacher, alltagsnaher Sprache zu erklären: warum der Miner gerade läuft oder gestoppt wurde, wie viel Solarstrom verfügbar ist und was die nächste Aktion bedeutet.
